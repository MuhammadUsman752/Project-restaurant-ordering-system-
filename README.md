#  Restaurant Ordering System

A Python  application built using **Object-Oriented Programming** concepts.

##  OOP Concepts Used
- **Class & Object** — FoodItem as base class
- **Inheritance** — MenuItem and SpecialMenuItem inherit from FoodItem
- **Polymorphism** — get_price() behaves differently in each class
- **Encapsulation** — Order class uses a private __cart list
## Features
- Browse full restaurant menu
- Special discounted items highlighted
- Add / remove items from cart
- Auto-calculated subtotal, tax, and grand total
- Clean bill popup at checkout
